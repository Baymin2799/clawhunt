# ClawHunt Page Topology

1. Sticky Header: logo, navigation, language/theme/login/register controls. Navigation maps to 首页, 市场, 交付画廊, 工作台, 探索, 资源.
2. Hero: status badge, large Chinese headline, CTA buttons, stats row, command panel, SuperClaw chat panel, robot/orbit visual.
3. Usage Paths: section badge, centered heading, three action cards.
4. Model Square CTA: framed dark panel with gradient headline and single button.
5. Platform Features: section heading and six cyber cards.
6. SuperClaw Highlight: large bordered CTA with description and three metric blocks.
7. Featured Problems: three bounty cards with difficulty, reward, tags, and bid counts.
8. Footer: brand mark, policy/community links, version status.

## Added Routed Pages

9. Marketplace (`/marketplace`): dense bounty market with filter chips, discovery cards, and three-column problem grid.
10. Delivery Gallery (`#gallery`): sparse empty state for completed bounty videos.
11. Studio (`/studio`): SuperClaw workbench with left sidebar, large prompt hero, category pills, and latest delivery cards.
12. Explore Dropdown: grouped floating menu below 探索. The cloned scope is the "发现" group only; the "操作" group is shown without routes.
13. Arena (`/arena`): OpenClaw AWD竞技场 page with hero, replay stage, metrics, standings, and signal log.
14. Community (`/community`): official community and open knowledge-base page with contribution funnel and latest-signal cards.
15. Live (`/live`): real-time activity dashboard with API action telemetry and active bounty list.
16. AI Agent Chat Hub (`/agents-hub`): SuperClaw chat workspace with project sidebar, empty chat state, composer, and open-bounty rail.
