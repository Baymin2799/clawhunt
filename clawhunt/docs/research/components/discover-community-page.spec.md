# DiscoverCommunityPage Specification

## Overview
- **Target file:** `src/App.tsx`
- **Screenshot:** `docs/design-references/discover/community-original-full.png`
- **Extraction:** `docs/research/clawhunt-discover-community-extraction.json`
- **Interaction model:** static community/knowledge-base landing page

## DOM Structure
- Full-width hero: official community badge, "爪寻AI开源知识库" headline, description, signal loop, three CTAs.
- Knowledge callout: entry to knowledge base and contribution path.
- Four-step process: 公开构建, 提交证据, 获得背书, 连接资本.
- Latest signal grid with six knowledge cards.

## Text Content
面向 AI Agent 开发者、VC 与企业需求方的官方社区：用真实任务验证产品，用开源证据沉淀背书，用社区 deal flow 连接开发者与资本。

## Responsive Behavior
- Desktop: four-step row and three-column knowledge grid.
- Mobile: all cards stack vertically.
